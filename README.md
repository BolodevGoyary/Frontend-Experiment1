# Frontend-Experiment1: Personal Profile Page (HTML Basics)

## Objective
To build proficiency in structuring content using HTML by creating a personal profile page that showcases basic information about yourself.

## Description
This experiment demonstrates the following:

- Displaying your **name** and a brief **bio**.
- Including a **profile picture**.
- Using **headings**, **paragraphs**, **unordered and ordered lists**.
- Adding at least **one link** to an external website (e.g., GitHub or university website).

## Files in this Repository
- `index.html` — The main HTML file containing the personal profile page.
- `profile.png` — Your personal profile image.

## How to Run
1. Clone or download this repository to your local machine.
2. Open `index.html` in any web browser (Chrome, Firefox, Edge, etc.).
3. You should see your personal profile page with your name, bio, skills, hobbies, and links.
